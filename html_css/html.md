## [1] HTML 이란

+ HTML은 현재의 웹 표준
  + 웹 표준 : 구조(HTML) + 디자인(CSS) + 기능(JS)로 명확히 구분해서 작성하도록 되어있음
+ HTML은 웹페이지를 작성(구조화)하기 위한 언어
  + 태그 등을 활용해 문서나 데이터의 구조를 명시하는 언어(Markup language)
